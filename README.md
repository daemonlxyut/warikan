# warikan

## Project setup
```sh
# cloning
git clone https://github.com/daemonlxyut/warikan.git
cd warikan

# install dependencies
yarn install

# launch local server
yarn serve

# compile and create output for deploying
yarn build

# lints and fixes files
yarn lint
```

## Creating Project

```sh
vue create warikan
cd warikan

vue add vuex
vue add vuetify
# -> create backup of App.vue
vue add router
# -> fix router using backup of App.vue
```
